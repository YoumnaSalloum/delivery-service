# Delivery-Service
## Prerequisites:
You must have Docker Installed in your System !
### clone the repo and run
```cd frontend```
```npm install```
```npm start```
### run Docker
```docker compose up```
```docker run -it -p 9001:8000 -v $(pwd):/app docker-node```
## The App should be App :

visit client : http://localhost:3000

visit server : http://localhost:8080
